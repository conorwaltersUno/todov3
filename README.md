## TODO V3.0

This is a simple TODO application that uses the following tech stack

- React frontend
- Express backend API
- PostrgresSQL database
- Prisma as a database ORM
- AWS S3 to host frontend
- AWS Lightsail to host API
- AWS RDS to host database
